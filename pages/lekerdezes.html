    Összes páciens neve

  • SELECT nev FROM paciensek;

    • Azok a páciensek, akiknek volt vizsgálatuk

  • SELECT név FROM paciensek WHERE pid IN (SELECT pid FROM vizsgalat);

    • Egy adott páciens (Kovács Krisztián) Pid-je

  • SELECT pid FROM paciensek WHERE nev = 'Kovács Krisztián';

    • Az összes diagnosztizált betegség

  • SELECT DISTINCT bnev FROM diagnózis;

    • Azok a betegségek, amelyekhez tartozik gyógyszer

  • SELECT bnev FROM mit_kap;

    • Azok a gyógyszerek, amelyek fel vannak írva valamilyen betegségre

  • SELECT gynev FROM gyogyszer WHERE gynev IN (SELECT gynev FROM mit_kap);

    Azok a páciensek, akik érzékenyek valamire

  • SELECT nev FROM paciensek WHERE pid IN ( SELECT pid FROM mire_erzekeny WHERE gynev IS NOT NULL );

    • Azok a páciensek, akik nem érzékenyek gyógyszerre

  • SELECT nev FROM paciensek WHERE pid NOT IN ( SELECT pid FROM mire_erzekeny WHERE gynev IS NOT NULL );

    • Egy adott páciens vizsgálatainak dátuma

  • SELECT datum FROM vizsgalat WHERE pid = ( SELECT pid FROM paciensek WHERE nev = 'Kovács Krisztián' );

    • Azok a betegségek, amelyek szerepelnek diagnózisban

  • SELECT bnev FROM betegseg WHERE bnev IN (SELECT bnev FROM diagnózis);
